Модуль D3.6 ИТОГОВОЕ ЗАДАНИЕ 
Группа FPW-57, Ширяев К.В.
Создана новую страницу с адресом templates\news\, на которой должен выводиться список всех новостей.
Выводятся все статьи в виде заголовка, даты публикации и первых 20 символов текста.
Новости  выводяться в порядке от более свежей к самой старой.(Сверху вниз)
Сделана отдельная страница для полной информации о статье /news/<id новости>.(Загружены три поста с id = 1; 2; 3; 5. Новость с id = 4 была удалена.)
На этой странице есть вся информация о статье. Название, текст и дата загрузки в формате день.месяц.год.
Написан собственный фильтр censor, который заменяет буквы нежелательных слов в заголовках и текстах статей на символ «*», применялся в статьях с id=3; 5).
Шаблон default.html используется как основа.
